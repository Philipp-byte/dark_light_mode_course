Grundlagen des Dark/Light Mode

Was ist Dark Mode?

Der Dark Mode ist eine Darstellungsoption, die helle Texte auf dunklem Hintergrund anzeigt. Er reduziert die Augenbelastung bei schlechten Lichtverhältnissen und kann auf OLED-Displays Akku sparen.

Was ist Light Mode?

Der Light Mode ist die traditionelle Darstellung mit dunklen Texten auf hellem Hintergrund. Er bietet gute Lesbarkeit bei Tageslicht und entspricht dem klassischen Druckdesign.

Barrierefreiheit

Ein gut implementierter Dark/Light Mode berücksichtigt Barrierefreiheit durch ausreichenden Kontrast zwischen Text und Hintergrund und respektiert die Systemeinstellungen der Benutzer.

Implementierung mit CSS-Variablen

:root {
    /* Light Mode (Standard) */
    --background-color: #ffffff;
    --text-color: #333333;
    --primary-color: #4361ee;
    --secondary-color: #f8f9fa;
    --accent-color: #f72585;
}

[data-theme="dark"] {
    /* Dark Mode */
    --background-color: #121212;
    --text-color: #e0e0e0;
    --primary-color: #4cc9f0;
    --secondary-color: #1e1e1e;
    --accent-color: #f72585;
}

body {
    background-color: var(--background-color);
    color: var(--text-color);
    transition: background-color 0.3s ease, color 0.3s ease;
}
<!-- Toggle-Switch für Dark/Light Mode -->
<div class="theme-switch-wrapper">
    <label class="theme-switch" for="checkbox">
        <input type="checkbox" id="checkbox" />
        <span class="slider round"></span>
    </label>
    <span id="theme-label">Light Mode</span>
</div>
// Event-Listener für den Toggle-Switch
document.querySelector('.theme-switch input[type="checkbox"]').addEventListener('change', function() {
    if (this.checked) {
        // Zu Dark Mode wechseln
        document.documentElement.setAttribute('data-theme', 'dark');
        document.getElementById('theme-label').textContent = 'Dark Mode';
        
        // Optional: Einstellung speichern
        localStorage.setItem('theme', 'dark');
    } else {
        // Zu Light Mode wechseln
        document.documentElement.setAttribute('data-theme', 'light');
        document.getElementById('theme-label').textContent = 'Light Mode';
        
        // Optional: Einstellung speichern
        localStorage.setItem('theme', 'light');
    }
});

Systemeinstellungen erkennen Optional

Moderne Webseiten können die Systemeinstellungen des Benutzers erkennen und automatisch den passenden Modus anwenden:

// Systemeinstellungen erkennen
if (window.matchMedia && window.matchMedia('(prefers-color-scheme: dark)').matches) {
    // System verwendet Dark Mode
    document.documentElement.setAttribute('data-theme', 'dark');
    document.querySelector('.theme-switch input[type="checkbox"]').checked = true;
    document.getElementById('theme-label').textContent = 'Dark Mode';
}

// Änderungen der Systemeinstellungen überwachen
window.matchMedia('(prefers-color-scheme: dark)').addEventListener('change', e => {
    if (e.matches) {
        // Wechsel zu Dark Mode
        document.documentElement.setAttribute('data-theme', 'dark');
        document.querySelector('.theme-switch input[type="checkbox"]').checked = true;
        document.getElementById('theme-label').textContent = 'Dark Mode';
    } else {
        // Wechsel zu Light Mode
        document.documentElement.setAttribute('data-theme', 'light');
        document.querySelector('.theme-switch input[type="checkbox"]').checked = false;
        document.getElementById('theme-label').textContent = 'Light Mode';
    }
});

Best Practices für Dark/Light Mode

Farbauswahl

  • Keine reinen Farben: Vermeide reines Schwarz (#000000) oder reines Weiß (#FFFFFF) für große Flächen.
  • Satte Farben reduzieren: Im Dark Mode sollten Farben weniger gesättigt sein, um Blendeffekte zu vermeiden.
  • Ausreichender Kontrast: Achte auf ein Kontrastverhältnis von mindestens 4.5:1 zwischen Text und Hintergrund.
  • Farbhierarchie beibehalten: Die visuelle Hierarchie sollte in beiden Modi konsistent sein.

Implementierung

  • CSS-Variablen nutzen: Verwende CSS-Variablen für alle Farben, um sie zentral zu verwalten.
  • Sanfte Übergänge: Füge Transitionen hinzu, um den Wechsel zwischen den Modi angenehm zu gestalten.
  • Systemeinstellungen respektieren: Berücksichtige die Präferenzen des Benutzers mit prefers-color-scheme.
  • Benutzereinstellungen speichern: Speichere die Auswahl des Benutzers im localStorage, um sie bei zukünftigen Besuchen wiederherzustellen.

Barrierefreiheit

  • Kontrast testen: Überprüfe den Kontrast in beiden Modi mit Tools wie dem Kontrastrechner.
  • Fokusindikatoren: Stelle sicher, dass Fokusindikatoren in beiden Modi gut sichtbar sind.
  • Bilder und Grafiken: Passe Bilder und Grafiken an den jeweiligen Modus an oder verwende neutrale Versionen.
  • Text-Schatten vermeiden: Verzichte auf Text-Schatten, die die Lesbarkeit beeinträchtigen können.
