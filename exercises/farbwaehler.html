CSS-Code

:root {
    /* Light Mode (Standard) */
    --background-color: #ffffff;
    --text-color: #333333;
    --primary-color: #0078d7;
    --secondary-color: #f5f5f5;
    --accent-color: #ff9e1b;
}

[data-theme="dark"] {
    /* Dark Mode */
    --background-color: #222222;
    --text-color: #f8f9fa;
    --primary-color: #4da6ff;
    --secondary-color: #444444;
    --accent-color: #ff9642;
}

Anleitung

  1. Wähle die Farben für den Light Mode aus
  2. Wähle die Farben für den Dark Mode aus
  3. Klicke auf "Vorschau anzeigen", um die Farben zu testen
  4. Kopiere den generierten CSS-Code für dein Projekt

Tipps für gute Farbschemata

  • Achte auf ausreichenden Kontrast zwischen Hintergrund- und Textfarbe (mindestens 4.5:1)
  • Verwende keine reinen Farben (#000000 oder #FFFFFF) für große Flächen
  • Wähle für den Dark Mode etwas hellere Primär- und Akzentfarben
  • Teste dein Farbschema auf Barrierefreiheit
